# Primeros pasos

Bienvenido al Accessibility Design Hub. Esta guía presenta la estructura de la documentación y cómo usar las demos interactivas.

## Qué encontrarás aquí

- **Guías** — Artículos breves sobre temas de accesibilidad (contraste de color, gestión del foco, semántica, etc.).
- **Demos interactivas** — Cada guía puede incluir componentes React en vivo para ver y probar patrones.
